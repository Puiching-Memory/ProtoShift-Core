# ProtoShift-Core

ProtoShift-Core 是一个面向 Godot 4 **原型开发** 的工作台

------------------------------------------------------------------

## 功能特色

你只需要负责**构思点子**, 我们来为你处理好以下事情:

### Lv. C 施工中
- 点子优化

### Lv. B 施工中
- 搭建一个可运行的原型游戏
- 跨平台(PC/移动端)和硬件适配(键鼠/手柄/VR)
- 性能优化
- 教程和提示

### Lv. A 暂无计划
- 多人联机
- 场景交互

### Lv. S 暂无计划
- 本地化
- 社区搭建
- 测试, QA, CI/CD
- 多媒体(美术/音效/动画/UI/叙事/演出/关卡设计)
- 合法性
- 商业化
- 项目管理

### Lv. S+ 暂无计划
- 完成一个游戏
- 获取资金
- 平衡工作和生活

-----------------------------------------------------------------

## Lv. C 点子优化

创意/想法/需求 -> 设计目的 -> 核心玩法 -> 总体流程(对开发) -> 视觉内容(美术/动画/UI) -> 数值设计 -> 奖励/成长 -> 教学引导 -> 落地验证
